# Klavye-Antrenman
C# Klavye Antrenmanı

3 seviyeden oluşmaktadır. Süre ve seviye geçme skorunu kendiniz belirleyebilirsiniz. "Enter" tuşu ile diğer harfe geçebilirsiniz. Her yanlış harf girişiniz de süreniz kısılmakta ve skorunuz düşmektedir.
